Conversion Algorithms

RGB

Classic (Median Cut)

The original algorithm. Averages all pixels in each block, generates a palette via Median Cut in RGB space, and maps each block to the nearest color. Fast and predictable — kept as a comparison baseline.

LAB

Classic LAB ✦ (Recommended)

Same clean solid-color blocks as Classic, with two key upgrades: palette generation runs in CIELAB color space for more perceptually distinct colors, and color matching uses LAB distance — the nearest color is the one that looks closest to the human eye, not just the one closest in RGB math. Best choice for authentic pixel art.

FS

LAB + Dithering

Adds Floyd-Steinberg dithering on top of the LAB pipeline. Color error from each block is diffused to neighbors in a 7/16 · 5/16 · 3/16 · 1/16 pattern, creating smooth gradients with a larger effective palette. Well suited for images with rich color transitions.

NNB

Tezumie Style

Inspired by the open-source Tezumie/Image-to-Pixel library. Uses nearest-neighbor center-pixel sampling instead of block averaging — preserving hard edges — then applies LAB Floyd-Steinberg dithering. Produces sharper outlines with a characteristic dithered texture.

Frequently Asked Questions

Is this pixel art tool free to use?

Yes! image2pixel is a free online tool for everyone. We also offer a full desktop version on Steam for professional users who need batch processing and more advanced features.

How do I get the best results?

Start with images that have clear subjects and high contrast. Avoid images with too much fine detail or noise, as they may look cluttered when pixelated.

What does the Palette option do?

The palette option forces the converter to use a specific set of colors. This is essential for creating art that looks like it belongs on specific vintage hardware like the Atari 2600 or the original GameBoy.

Which perler bead brands are supported?

We currently support 7 built-in fuse bead palettes from popular brands: Perler, Perler Mini, Hama Midi, Hama Mini, Nabbi, Artkal S (5mm), and Artkal C (2.6mm). Click the palette button in the result panel and scroll down to the "Fuse Bead Palettes" section to choose a brand.

How do I export a perler bead blueprint?

After uploading an image and selecting a fuse bead palette (e.g., Perler or Hama), a Blueprint checkbox appears in the result panel header. Check it, then click Save. The downloaded PNG will be a numbered grid where each cell shows the bead's official color code, perfect for following while building your project.

How do I save a true 1:1 pixel art file?

Check the 1:1 checkbox in the result panel, then click Save. The downloaded PNG uses one physical pixel per color block—ideal for game sprites, further upscaling, or importing into other pixel tools. The on-screen preview stays large for easier viewing.
