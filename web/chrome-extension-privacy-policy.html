Image2Pixel Picker Privacy Policy

Last updated: 2026-03-17

Image2Pixel Picker is a Chrome extension that converts a user-selected image on the current webpage into pixel art.

What the extension accesses

The extension accesses webpage images only after the user explicitly starts one of these actions:

  • clicks the extension popup and chooses Select Image
  • right-clicks an image and chooses the conversion menu item

The extension may temporarily access the selected tab, the selected image element, the selected image URL, and the saved conversion settings required to complete the user-requested conversion.

Data collection and transmission

Image2Pixel Picker does not create user accounts, collect personal identity information, sell user data, or use analytics and advertising trackers.

Image conversion runs locally in the browser. If the extension needs to read image bytes from the selected image URL, it fetches that image directly in the extension context only to process the image chosen by the user.

Data storage

The extension stores only the user's conversion settings in Chrome local storage, including pixel size and color count.

Permissions explanation

  • activeTab: used to access the current page only after user action
  • contextMenus: used to add the right-click conversion action
  • scripting: used to inject the conversion runtime only when the user triggers it
  • storage: used to remember conversion settings
  • <all_urls>: used to read or fetch the exact image selected by the user on supported webpages
